Core Endpoints

6 Methods Available
GET /api/{version}/book
Returns a summary of a specific book, including an array of its available chapters. Requires the name query parameter.
"book": "Genesis", "chapters": [ "1", "2", "3", "..." ]
GET /api/{version}/chapter
Retrieves all verses within a selected book's chapter. Requires book and chapter query parameters.
"book": "Genesis", "chapter": 1, "verses": [ { "verse": "1", "text": "In the beginning God created the heaven and the earth." } ]
GET /api/{version}/verse
The core text delivery method. Fetches the exact scriptural content for a specifically targeted single verse.
"book": "Genesis", "chapter": 1, "verse": 1, "text": "In the beginning God created the heaven and the earth."
GET /api/{version}/search
Perform a case-insensitive, globally scoped text search across the specified Bible version. Requires the q query parameter.
"results": [ { "book": "Deuteronomy", "chapter": 32, "verse": 20, "text": "And he said, I will hide my face from them..." } ]
GET /api/{version}/books
Lists all available standard books currently accessible inside the evaluated payload of the specified version.
"books": [ "Genesis", "Exodus", "Leviticus", "..." ]
GET /api/{version}/random
Explore serendipitously. Returns a completely randomized verse pulled instantly from anywhere within the dataset.
"book": "Psalms", "chapter": 23, "verse": 1, "text": "The LORD is my shepherd; I shall not want."
