The high-performance,

Bible data layer.

Scripture API provides a globally distributed, low-latency REST API for accessing multi-translation biblical texts with millisecond precision. Built for modern apps.

API Reference → View Supported Versions

Auto-Detected Translations

The system dynamically scans the datasets available on the server. To query the API, replace {bible} in the endpoints with one of the identifiers below.

Core Endpoints

6 Methods Available
GET /api/versions
Returns/Detects available bible api Available. 

  GET /api/versions
  

  {
  "versions":
    [
      "KJbible",
      "NIVbible"
    ]
  }
GET /api/metadata
provides information about the API itself 

  GET /api/metadata
GET /api/health
report whether your API service is alive and functioning correctly. 

  GET /api/health
  

    {
      "status": "ok",
      "timestamp": "2026-04-01T15:45:12.345Z",
      "bible_versions": ["KJbible", "NIVbible"],
      "api_version": "1.0.0",
      "database": "connected"
    }
GET /api/{bible}/summary
This response gives all bible navigation metadata in one cal 

  GET /api/KJbible/books/summary
  

    {
    "books": [
      {
        "name": "Genesis",
        "chapters": 50,
        "verses": 1533,
        "chapterSummary": [
          { "chapter": 1, "verses": 31 },
          { "chapter": 2, "verses": 25 },
          { "chapter": 3, "verses": 24 }
        ]
      }
    ]
  }
GET /api/{bible}/summary/{book}
single-book summary endpoint. 

  GET /api/KJbible/summary/Genesis
  

    {
    "book": "Genesis",
    "chapters": 50,
    "totalVerses": 1533,
    "chapterSummary": [
      { "chapter": 1, "verses": 31 },
      { "chapter": 2, "verses": 25 },
      { "chapter": 3, "verses": 24 }
    ]
  }
GET /api/{bible}/summary/{book}/{chapter}
This endpoint returns a chapter summary 

  GET /api/KJbible/summary/Genesis/1
  

    {
    "book": "Genesis",
    "chapter": 1,
    "verses": 31
  }
GET /api/{bible}/book
Returns metadata for a specific book and the list of its chapters. Requires the name query parameter. 

GET /api/KJbible/book?name=Genesis


{
  "book": "Genesis",
  "chapters": [1,2,3,4,5,6,"..."]
}
GET /api/{bible}/chapter
Returns all verses inside a specific chapter. Requires book and chapter. 

GET /api/KJbible/chapter?book=Genesis&chapter=1


{
  "book": "Genesis",
  "chapter": 1,
  "verses": [
    {
      "verse": 1,
      "text": "In the beginning God created the heaven and the earth."
    },
    {
      "verse": 2,
      "text": "And the earth was without form, and void..."
    }
  ]
}
GET /api/{bible}/verse
Fetch a single verse from the selected Bible translation. 

GET /api/KJbible/verse?book=Genesis&chapter=1&verse=1


{
  "book": "Genesis",
  "chapter": 1,
  "verse": 1,
  "text": "In the beginning God created the heaven and the earth."
}
GET /api/{bible}/search
Performs a case-insensitive search across the Bible text. Requires the q query parameter. 

GET /api/KJbible/search?q=for god so loved


{
  "results": [
    {
      "book": "John",
      "chapter": 3,
      "verse": 16,
      "text": "For God so loved the world, that he gave his only begotten Son..."
    }
  ]
}
GET /api/{bible}/books
Returns all books available in the selected Bible version. 

GET /api/KJbible/books


{
  "books": [
    "Genesis",
    "Exodus",
    "Leviticus",
    "Numbers",
    "Deuteronomy",
    "..."
  ]
}
GET /api/{bible}/random
Returns a randomly selected verse from the dataset. 

GET /api/KJbible/random


{
  "book": "Psalms",
  "chapter": 23,
  "verse": 1,
  "text": "The LORD is my shepherd; I shall not want."
}
GET /api/{bible}/passage
returns a range of verses from the same chapter. 

/api/KJbible/passage?book=John&chapter=3&start=16&end=18


{
"book": "Psalms",
"chapter": 23,
"verse": 1,
"text": "The LORD is my shepherd; I shall not want."
}
GET /api/{bible}/daily
Return one fixed verse per day.Every request that day returns the same verse 

/api/KJbible/daily


  {
   "date": "2026-04-01",
   "book": "Proverbs",
   "chapter": 3,
   "verse": 5,
   "text": "Trust in the LORD with all thine heart..."
  }
