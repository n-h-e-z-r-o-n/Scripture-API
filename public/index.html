Core Endpoints

6 Methods Available
GET /api/versions
Lists all available Bible translation datasets loaded on the server. 

  GET /api/versions
  

  {
  "versions":
    [
      "KJbible",
      "NIVbible"
    ]
  }
GET /api/metadata
Returns system-level metadata and configuration for the API. 

  GET /api/metadata
GET /api/health
Checks if the API engine is alive and reports system uptime. 

  GET /api/health
  

    {
      "status": "ok",
      "timestamp": "2026-04-01T15:45:12.345Z",
      "bible_versions": ["KJbible", "NIVbible"],
      "api_version": "1.0.0",
      "database": "connected"
    }
GET /api/{bible}/summary
Aggregates metadata for the entire specified Bible translation. 

  GET /api/KJbible/books/summary
  

    {
    "books": [
      {
        "name": "Genesis",
        "chapters": 50,
        "verses": 1533,
        "chapterSummary": [
          { "chapter": 1, "verses": 31 },
          { "chapter": 2, "verses": 25 },
          { "chapter": 3, "verses": 24 }
        ]
      }
    ]
  }
GET /api/{bible}/summary/{book}
Returns total chapters and verse configurations for a specific book. 

  GET /api/KJbible/summary/Genesis
  

    {
    "book": "Genesis",
    "chapters": 50,
    "totalVerses": 1533,
    "chapterSummary": [
      { "chapter": 1, "verses": 31 },
      { "chapter": 2, "verses": 25 },
      { "chapter": 3, "verses": 24 }
    ]
  }
GET /api/{bible}/bookOrder
Returns the chronological order of books for the specified translation. 

  GET  /api/KJbible/bookOrder
  

    {
  "books": [
    "Genesis",
    "Exodus",
    "Leviticus",
    "Numbers",
    "Deuteronomy",
    "Joshua",
     ....
GET /api/{bible}/summary/{book}/{chapter}
Provides verse count and metadata for a specific chapter. 

  GET /api/KJbible/summary/Genesis/1
  

    {
    "book": "Genesis",
    "chapter": 1,
    "verses": 31
  }
GET /api/{bible}/book
Fetches metadata and an overview of chapters for the specified book. 

GET /api/KJbible/book?name=Genesis


{
  "book": "Genesis",
  "chapters": [1,2,3,4,5,6,"..."]
}
GET /api/{bible}/chapter
Retrieves all verses within a specific chapter. 

GET /api/KJbible/chapter?book=Genesis&chapter=1


{
  "book": "Genesis",
  "chapter": 1,
  "verses": [
    {
      "verse": 1,
      "text": "In the beginning God created the heaven and the earth."
    },
    {
      "verse": 2,
      "text": "And the earth was without form, and void..."
    }
  ]
}
GET /api/{bible}/verse
Fetches the exact text of a specified verse. 

GET /api/KJbible/verse?book=Genesis&chapter=1&verse=1


{
  "book": "Genesis",
  "chapter": 1,
  "verse": 1,
  "text": "In the beginning God created the heaven and the earth."
}
GET /api/{bible}/search?q={query}
Executes a global text search across the specified translation. 

  GET /api/KJbible/search?q=faith
  GET /api/NIVbible/search?q=hope
  GET /api/KJbible/search?q=for god so loved


{
  "results": [
    {
      "book": "John",
      "chapter": 3,
      "verse": 16,
      "text": "For God so loved the world, that he gave his only begotten Son..."
    }
  ]
}
GET /api/{bible}/topic/{keyword}
Retrieves a curated list of verses relating to a specific topic or theme. 

  GET /api/KJbible/topic/faith
  GET /api/KJbible/topic/love
  GET /api/NIVbible/topic/hope


  {
    "version": "KJbible",
    "topic": "faith",

      { "book": "Matthew", "chapter": 17, "verse": 20, "text": "If ye have faith as a grain of mustard seed..." }
    ]
  }

GET /api/KJbible/books

/api/KJbible/passage?book=John&chapter=3&start=16&end=18

GET /app/api/[bible]/ref/[reference]
Parses human-readable strings (e.g., "John 3:16") into verse text. 

Single verse: GET /api/KJbible/ref/John 3:16
Verse range (same chapter): GET /api/KJbible/ref/John 3:16-18
Multi-chapter range: GET /api/KJbible/ref/Genesis 1:31-2:3


  {
    "version": "KJbible",
    "book": "Genesis",
    "start": { "chapter": 1, "verse": 31 },
    "end": { "chapter": 2, "verse": 3 },
    "verseCount": 4,
    "verses": [
      { "chapter": 1, "verse": 31, "text": "And God saw everything that he had made..." },
      { "chapter": 2, "verse": 1, "text": "Thus the heavens and the earth were finished..." },
      { "chapter": 2, "verse": 2, "text": "And on the seventh day God ended his work..." },
      { "chapter": 2, "verse": 3, "text": "And God blessed the seventh day..." }
    ]
  }
GET /api/{bible}/daily
Return one fixed verse per day.Every request that day returns the same verse Returns the Verse of the Day for the specified Bible translation. This is a pre-determined verse based on the current date and can be used for daily devotionals or app notifications. 

/api/KJbible/daily


  {
   "date": "2026-04-01",
   "book": "Proverbs",
   "chapter": 3,
   "verse": 5,
   "text": "Trust in the LORD with all thine heart..."
  }
