Bible API

A fast, lightweight, and extensible serverless REST API for Bible verses.

Introduction

Welcome to the Bible API. This API allows developers to easily fetch Bible books, chapters, and verses. The architecture is explicitly designed to be dynamic and to support multiple Bible versions (e.g. KJbible, NIVbible, ESVbible) natively as datasets are added.

Endpoints

GET /api/{version}/book

Retrieve a summary of a specific book, including its available chapters.

Query Parameters:

  • name (string) - The name of the book. Example: Genesis
GET /api/KJbible/book?name=Genesis
Example Response (200 OK): 
{
  "book": "Genesis",
  "chapters": ["1", "2", "3", "4", "5", "..."]
}

GET /api/{version}/chapter

Retrieve all verses within a specific chapter.

Query Parameters:

  • book (string) - The name of the book. Example: Genesis
  • chapter (number) - The chapter number. Example: 1
GET /api/KJbible/chapter?book=Genesis&chapter=1
Example Response (200 OK): 
{
  "book": "Genesis",
  "chapter": 1,
  "verses": [
    {
      "verse": "1",
      "text": "In the beginning God created the heaven and the earth."
    },
    ...
  ]
}

GET /api/{version}/verse

Retrieve a single, specific verse.

Query Parameters:

  • book (string) - The name of the book.
  • chapter (number) - The chapter number.
  • verse (number) - The verse number.
GET /api/KJbible/verse?book=Genesis&chapter=1&verse=1
Example Response (200 OK): 
{
  "book": "Genesis",
  "chapter": 1,
  "verse": 1,
  "text": "In the beginning God created the heaven and the earth."
}

GET /api/{version}/search

Search all verses in the provided version for a specific term or phrase. Results are capped at 100.

Query Parameters:

  • q (string) - The search query term. Example: faith
GET /api/KJbible/search?q=faith
Example Response (200 OK): 
{
  "results": [
    {
      "book": "Deuteronomy",
      "chapter": 32,
      "verse": 20,
      "text": "And he said, I will hide my face from them..."
    },
    ...
  ]
}

GET /api/{version}/books

List all available books in the specified version. (Optional Tooling Endpoint)

GET /api/KJbible/books
Example Response (200 OK): 
{
  "books": ["Genesis", "Exodus", "Leviticus", ...]
}

GET /api/{version}/random

Retrieve a random verse from any book or chapter.

GET /api/KJbible/random
Example Response (200 OK): 
{
  "book": "Psalms",
  "chapter": 23,
  "verse": 1,
  "text": "The LORD is my shepherd; I shall not want."
}

Error Handling

If an invalid parameter or missing version is provided, the API will respond with appropriate standard HTTP status codes combined with clear error messages.

{
  "error": "Bible version 'NIVbible' not found."
}
